============
Installation
============


Stable release
--------------

To install OverLINCS Git, run this command in your terminal:

.. code-block:: console

    $ pip install overlincs-git

This is the preferred method to install OverLINCS Git, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for OverLINCS Git can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/balouf/overlincs-git

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/balouf/overlincs-git/tarball/main

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ pip install .


.. _Github repo: https://github.com/balouf/overlincs-git
.. _tarball: https://github.com/balouf/overlincs-git/tarball/main
